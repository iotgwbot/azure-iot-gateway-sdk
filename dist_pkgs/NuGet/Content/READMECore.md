.NET NuGet Core Package for Windows Desktop
===========================================

Overview
--------

This NuGet package contains all of the core native (C++) binaries for the Azure IoT Gateway.

1. A sample native (C++) gateway executable (gw.exe) file.
2. The native (C++) binding layer interfaces for .NET, Java, and NodeJs (C++). Which includes `gateway.dll`, `aziotsharedutil.dll`, `nanomsg.dll`, `dotnet.dll`, `java_module_host.dll`, `node.dll`, `nodejs_binding.dll`.
